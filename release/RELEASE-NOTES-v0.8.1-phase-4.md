# Strategos v0.8.1 — Stabilization Phase 4

## PWA and Accessibility

This phase establishes the first reliable offline shell and accessibility baseline.

### Included

- versioned service worker;
- offline application shell after the first successful load;
- automatic cache replacement between releases;
- reduced-motion support;
- accessible icon-button names;
- semantic settings switches;
- focus management after route changes;
- bounded timer and phase announcements;
- expanded web-app manifest metadata.

### Identity Sound

No provisional opening sound was added. The future Arete Chime remains a separate original brand asset.
