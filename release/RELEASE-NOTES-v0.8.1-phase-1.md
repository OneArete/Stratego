# Strategos v0.8.1 — Stabilization and State Integrity

## Phase 1

This package implements the first stabilisation slice directly on the v0.8.0 baseline.

### Included

- versioned state schema;
- v0.8.0 to v0.8.1 migration;
- Legacy to Recovery migration;
- soreness safety guard;
- emotional-load integration;
- explicit Agora caution record;
- qualitative Current Judgement confidence;
- behavioural tests.

### Not yet included

- timestamp-based practice execution;
- interrupted-session resume;
- state export and import;
- service worker;
- complete accessibility corrections.

Those remain in the frozen v0.8.1 scope and will be implemented in the next slices.
