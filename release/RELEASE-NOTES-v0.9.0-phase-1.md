# Strategos v0.9.0 — Phase 1

## Evidence Diversity and Memory Integrity

This phase begins the v0.9.0 development cycle from the official v0.8.1 baseline.

Implemented:

- evidence-family records for Advisor positions;
- evidence-diversity assessment;
- correlated-evidence caution;
- explicit contradiction detection;
- confidence penalties for narrow evidence and unresolved tensions;
- structured Advisor memory weights;
- observation counts and positive, neutral and negative evidence;
- recency decay with a 90-day half-life;
- candidate-learning status;
- v0.8.1 to v0.9.0 state migration;
- person-facing evidence-diversity and contradiction sections.

No external AI, cloud account or provisional Arete Chime audio is included.
