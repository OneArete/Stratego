# Discovery — Advisor Memory Belongs to the Specialist

## Observation
A global personalization profile would make independent Advisors converge prematurely.

## Decision
Experience belongs to each specialist. Strategos synthesizes but does not own specialist memory.

## Open questions
- How should contradictory learning decay over time?
- What evidence diversity is required before an Advisor calls something a pattern?
- Which memories require explicit human confirmation before influencing judgement?
