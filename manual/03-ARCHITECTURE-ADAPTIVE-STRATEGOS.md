# Volume III — Adaptive Strategos Architecture

## Learning loop
Signals → Understanding → Advisor Memory → Independent Advisor Reasoning → Agora → Current Judgement → Explanation → Human Choice → Practice → Reflection → Advisor Learning.

## Memory boundary
Each Advisor owns experience, bounded practice-weight adjustments, confidence, coverage and notes. No Advisor can directly edit another Advisor's memory. Strategos reads all memories only during synthesis.

## Understanding Coverage
Coverage represents the amount and diversity of reflected evidence available to a specialist. It is not certainty, truth or a score of the person.

## Monthly Council
The Monthly Council is a retrospective synthesis. It reports what each Advisor learned, where coverage is weak and how the overall understanding changed. It does not issue a new practice judgement.
